# 🍽️ Node.js Restaurant Website with GitHub Actions

A responsive **Restaurant Website** built using **Node.js, Express.js, HTML, CSS, and JavaScript** with automated workflow checks using **GitHub Actions**.

This project demonstrates how to build and run a Node.js web application while using GitHub Actions for automation on every push.

---

## 🚀 Live Project Overview

This project includes:

✅ Responsive restaurant website UI  
✅ Node.js backend server  
✅ Static files served from `public/` folder  
✅ Automated GitHub Actions workflow  
✅ Version control using Git & GitHub

---
